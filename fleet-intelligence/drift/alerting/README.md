# Drift Alerting

Alert rules and notification routing for drift detection signals.

## 📂 Components

| Component           | Description                                                   |
|---------------------|---------------------------------------------------------------|
| Threshold evaluator | Compares drift metrics against configurable limits            |
| Notification router | Routes alerts to Grafana, Azure Monitor, or webhook endpoints |
| Escalation policy   | Progressive severity based on drift duration and magnitude    |
