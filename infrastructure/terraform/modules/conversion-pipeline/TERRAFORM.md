ADLS Gen2 storage account that backs the raw → converted ingest pipeline.
Hierarchical namespace is required so Microsoft Fabric can mount converted
data via OneLake shortcuts.

## Requirements

| Name      | Version         |
|-----------|-----------------|
| terraform | >= 1.9.8, < 2.0 |
| azurerm   | >= 4.51.0       |
| fabric    | ~> 1.0          |
